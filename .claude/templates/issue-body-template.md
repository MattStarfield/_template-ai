# Issue Body Template

**Version**: 1.0.0
**Purpose**: Standard GitHub issue body format.

## Template

```markdown
## Overview

[Clear description of what this issue is about]

## Requirements

- [ ] [Requirement 1]
- [ ] [Requirement 2]

## Acceptance Criteria

- [ ] [Criterion 1 - specific, testable]
- [ ] [Criterion 2 - specific, testable]

## Technical Notes

[Implementation hints, constraints, context]

### Affected Components
- [ ] `component/path` - [impact description]

---

**Complexity**: Low | Medium | High
**Dependencies**: [List or "None"]
```
