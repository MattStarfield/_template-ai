# Guide: Re-ordering PCBAs from Seeed Studio Fusion

Files used for PCBA ordering can be found [**here**](hardware/elec/eagle_v9-6-2/SYG-ASM-001_SygnalAir_PCB/seeed_fusion_pcba_order_info)

1. Visit [**Seeed Studio Fusion > PCB/PCBA order page**](https://www.seeedstudio.com/fusion_pcb.html)
      * Log-in or Create a User Account

1. Upload [<img src="/img/zip_icon.png" alt=".jpg" width="15">**Gerber Files (.zip)**](hardware/elec/eagle_v9-6-2/SYG-ASM-001_SygnalAir_PCB/seeed_fusion_pcba_order_info/GERBERS_SYG-ASM-001_2021-08-20.zip)

1. Fill out PCB options as below (adjust **_PCB Quantity_** as needed)<br><img src="/img/pcb_order_form.png" alt="pcb_order_form.png" width="600">

1. Expand the **PCB Assembly** section

1. Upload [<img src="/img/zip_icon.png" alt=".jpg" width="15">**Assembly Files (.zip)**](hardware/elec/eagle_v9-6-2/SYG-ASM-001_SygnalAir_PCB/seeed_fusion_pcba_order_info/ASSEMBLY_SYG-ASM-001_2021-08-17.zip)

1. Upload [<img src="/img/gsheets_icon.png" alt="gsheets_icon.png" width="15">**Seeed Formatted BOM File (.xlsx)**](hardware/elec/eagle_v9-6-2/SYG-ASM-001_SygnalAir_PCB/seeed_fusion_pcba_order_info/seeed_pcba_bom_2021-08-17.xlsx)

1. Adjust **_PCBA Qty_** as needed (typically to match **_PCB Quantity_**)

1. Select **Visual Inspection** since we do not have a _Functional Testing Procedure_<br><img src="/img/pcb_assembly_form.png" alt="pcb_assembly_form.png" width="600">

1. Review order information and click **Add to Cart**<br><img src="/img/pcb_order_add_to_cart.png" alt="pcb_order_add_to_cart.png" width="400">

1. Seeed Studio staff will review the order to ensure they have all of the required information. They will respond within 2-3 business days to confirm the order, or request more information.

1. Order progress can be tracked in<br>
**[Account Profile] > Order History > [Order #] > View Details**<br><img src="/img/seeed_order_details.png" alt="seeed_order_details.png" width="600">

1. Once an order has been placed, you can use the **Reorder** button to easily order more PCBAs<br><img src="/img/seeed_reorder.png" alt="seeed_reorder.png" width="600">
